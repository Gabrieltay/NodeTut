# NodeTut

### Installation 

Install NodeJs and Npm using Homebrew. Start a Node server.
```sh
$ brew install node
$ cd NodeTut
$ npm install -d
$ node index
```

Hit the [server](http://localhost:3000/) to see if its working


Expected result:
```json
{
    "Message": "Hello World"
}
```
